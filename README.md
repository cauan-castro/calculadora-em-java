# 🧮 Calculadora em Java

Pequeno projeto de calculadora desenvolvido em **Java** com o objetivo de praticar e aprimorar meus conhecimentos na linguagem.

Com este projeto, você poderá realizar cálculos diretamente pelo **terminal**, ou executá-lo em sua **IDE favorita** para estudar e modificar o código.

---

## 🚀 Funcionalidades

- Operações matemáticas básicas via terminal  
- Execução simples e rápida  
- Código aberto para estudo e modificações  

---

## 📋 Requisitos

Antes de começar, você precisa ter instalado em sua máquina:

- ☕ Java JRE ou Java JDK

---

## 📁 Arquivos do Projeto

- `calculadora.class` → Para execução direta via terminal  
- `calculadora.java` → Para edição e execução em uma IDE  

---

## ▶️ Como Executar

### 🖥️ Via IDE

1. Execute normalmente o projeto
2. faças as modificações que desejar

### 💻 Via Terminal

1. Abra o terminal na pasta do projeto  
2. Execute o comando:


```bash
java calculadora
